# SkillForge AI 🚀

Build Your Future With AI-Powered Learning

SkillForge AI is a modern frontend-based career development platform designed to help students and beginners explore technology roadmaps, discover project ideas, improve their resumes, and prepare for industry careers.

## 🌟 Features

### Career Roadmaps

* Frontend Developer Roadmap
* Backend Developer Roadmap
* Full Stack Developer Roadmap
* AI Engineer Roadmap
* Data Scientist Roadmap
* Cybersecurity Analyst Roadmap
* Cloud Engineer Roadmap

### Technology Explorer

Detailed learning guides for:

* HTML
* CSS
* JavaScript
* React
* Node.js
* Python
* MySQL
* Git & GitHub
* Flutter
* AI Tools

Each technology includes:

* Overview
* Skills to Learn
* Real-World Applications
* Project Ideas
* Career Paths
* Interview Questions
* Official Learning Resources

### AI Project Idea Generator

Generate personalized project recommendations based on:

* Skill Level
* Technologies Known
* Interests
* Career Goals
* Available Learning Time

### Resume, Portfolio & GitHub Hub

* ATS Resume Checklist
* Portfolio Checklist
* GitHub Profile Checklist
* Common Recruiter Rejection Reasons

### Student-Friendly Features

* Dark Mode
* Responsive Design
* Search Functionality
* Interactive Modals
* Career Filters
* Scroll Animations
* Progress Indicators
* Contact Form Validation
* Website Rating System
