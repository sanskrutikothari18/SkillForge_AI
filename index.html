Resume, Portfolio & GitHub

Practical, recruiter-tested checklists for building a profile that actually gets shortlisted.

📄 Resume Checklist

  • ATS friendly (no tables / columns / images)
  • Strong Skills section grouped by category
  • Projects section with links & impact metrics
  • Education with relevant coursework
  • Certifications from trusted platforms
  • 1 page, action verbs, no fluff

🎨 Portfolio Checklist

  • Clear About Me with photo & focus area
  • 3–5 polished, deployed Projects
  • Working Contact Form or email link
  • Fully Responsive Design on mobile
  • Fast load time & accessible colors
  • Custom domain + favicon

🐙 GitHub Checklist

  • Profile README with intro, stack & links
  • Pinned Projects (6 best repos)
  • Proper Documentation in every README
  • Meaningful Commits (not "update")
  • Live demo + screenshots in repo
  • Green contribution graph

🚫 Why Recruiters Reject Profiles

  • Generic resume not tailored to the JD
  • Projects without live demos or GitHub links
  • Empty / inactive GitHub profile
  • Spelling and grammar mistakes
  • Listing skills you can't explain
  • No measurable impact (numbers, results)
  • Cluttered design with multiple fonts/colors
  • Outdated tech stack & no learning evidence
